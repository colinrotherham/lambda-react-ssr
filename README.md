React SSR + Lambda demo
=======================

### What is it?
Proof of concept React SSR (server-side rendered HTML) via AWS Lambda

### Initial setup

Install Node.JS v13.2+

1. Install

```console
npm ci
```

2. Build

```console
npm run build
```

3. Preview

Run via a local Node.js server

```console
npm start
```

4. Deploy

Upload **package.zip** to your Lambda or via AWS CLI
